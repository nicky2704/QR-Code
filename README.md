A simple and responsive QR Code Generator that converts text or URLs into scannable QR codes instantly. This project provides a fast and user-friendly way to generate QR codes for websites, links, and other text-based data.

Features:
Generate QR codes instantly
Convert URLs and text into scannable QR codes
Real-time QR code preview
Lightweight and easy to use

Technologies Used:
CSS3
Python
QR Code Libraries

Project Objective:
The goal of this project is to simplify sharing information by generating QR codes that can be scanned using mobile devices. It demonstrates the practical use of QR technology and basic web development concepts.
